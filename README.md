# vue3-framework7-capacitor3-template
该脚手架基于Vue3.0 + Framework7(v6.0) + Capacitor3.0构建
其中内置了Jpush极光推送，开发者打包时需要将APPkey 替换成自己的。
## Project setup
安装依赖
```
npm install
```

### 运行项目
```
npm run serve
```

### 构建生产环境
```
npm run build-prod
```

### 当添加了新插件后，运行
```
npm run sync
```
### 同步代码至Native Platform
```
npm run copy-prod
```

### Lints and fixes files
```
npm run lint
```
